# ServiceNow Environment Switcher

ServiceNowの複数環境を簡単に切り替えるChrome拡張機能

## 🎯 機能

- **環境登録**: Dev/Test/Prod など複数の環境を登録
- **ワンクリック切り替え**: 登録済み環境に瞬時に移動
- **色分け表示**: 環境ごとに色を設定して視覚的に区別
- **現在の環境を表示**: 開いている環境を強調表示
- **動的アイコン**: タブごとにアイコンの色が環境に応じて変化
- **バッジ表示**: 拡張機能アイコンに環境名を表示

## 📸 スクリーンショット

### メイン画面
環境の登録と一覧表示

### 現在の環境の表示
開いている環境が緑色で強調表示され、「現在の環境」バッジが表示されます

### アイコンの色変更
環境に応じて拡張機能のアイコンの色が変わります

## 🚀 インストール方法

### Chrome Web Storeから（公開後）
*準備中*

### 開発版として手動インストール

1. このリポジトリをクローン
```bash
git clone https://github.com/gozu52/chrome_extention_environment-switcher.git
```

2. Chromeで `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」をONにする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. クローンした `environment-switcher` フォルダを選択

## 📖 使い方

### 1. 環境を登録

1. 拡張機能アイコンをクリック
2. 「環境を追加」セクションで：
   - 環境名を入力（例: Dev, Test, Prod）
   - ServiceNowのURLを入力
   - 色を選択
3. 「環境を追加」ボタンをクリック

### 2. 環境を切り替え

1. 拡張機能アイコンをクリック
2. 「登録済み環境」から移動したい環境をクリック
3. 自動的に該当のURLに移動

### 3. 環境を削除

1. 拡張機能アイコンをクリック
2. 削除したい環境の「削除」ボタンをクリック

## 💡 推奨される使い方

### 色分けの例

- **Dev環境**: 緑色 🟢 - 開発環境
- **Test環境**: 黄色 🟡 - テスト環境
- **Prod環境**: 赤色 🔴 - 本番環境（注意！）
- **Sandbox環境**: 青色 🔵 - 検証環境

## 🛠 技術スタック

- **Manifest V3**: 最新のChrome拡張機能仕様
- **Vanilla JavaScript**: 軽量でシンプル
- **Chrome Storage API**: 環境設定の永続化
- **Canvas API**: 動的アイコン生成
