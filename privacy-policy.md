# Privacy Policy for ServiceNow Environment Switcher

**Last updated: January 12, 2026**

## Overview
ServiceNow Environment Switcher is a Chrome extension that helps you manage and switch between multiple ServiceNow environments efficiently.

## Data Collection
This extension does **NOT collect, transmit, or share any personal data**.

## Data Storage
All data is stored **locally** in your browser using Chrome Storage API:
- Environment names
- Environment URLs
- Color preferences
- Group names
- Access history (timestamps and counts)

**No data leaves your device.**

## Permissions
This extension requires the following permissions:

### storage
To save your environment settings locally in your browser.

### tabs
To switch between ServiceNow environments and detect the current environment.

### activeTab
To identify which ServiceNow environment is currently open.

## Third-Party Services

This extension does **NOT** use any third-party services, analytics, or tracking tools.

## Data Security

- All data is stored locally using Chrome's secure storage API
- No data is transmitted to external servers
- No authentication or login is required

## Updates

This privacy policy may be updated from time to time. The "Last updated" date at the top of this page will be revised accordingly.

## Contact

For questions or concerns about this privacy policy, please contact:
- GitHub: https://github.com/gozu52/chrome_extention_environment-switcher/issues


## Your Rights

You can delete all stored data at any time by:
1. Removing the extension from Chrome
2. Or clearing your browser's storage for this extension

**This extension respects your privacy and operates entirely offline.**
